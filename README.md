# homebrew-tap

Homebrew tap for [sequencestream](https://github.com/sequencestream) tools.

## Install

```bash
brew install sequencestream/tap/c3
```

Or tap first, then install:

```bash
brew tap sequencestream/tap
brew install c3
```

## Available formulae

- **c3** — [Code Creative Center](https://github.com/sequencestream/c3), harness/loop engineering for AI software work.
